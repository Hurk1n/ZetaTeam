About

战队简介

Zeta 战队是由杭州电子科技大学信息工程学院学生组建的专业技术团队，面向对网络安全技术感兴趣的同学，专注于网络安全技术研究与实践，主要参与 CTF（Capture The Flag）竞赛及相关安全活动。

战队由热爱网络安全的在校学生组成，致力于提升成员的安全技能、团队协作与竞赛水平。研究方向覆盖 Web 安全、二进制安全、软件逆向、密码学与杂项等核心方向。无论是初学者还是已有一定基础的爱好者，只要对网络安全抱有热情，我们都欢迎你的加入。

团队通过日常训练、技术分享与赛事实践，推动校园网络安全氛围建设，并为成员积累可迁移的专业能力与竞赛经验。

什么是 CTF

CTF 是网络安全领域常见的技术竞技形式，起源于 DEFCON 全球黑客大会。题目将安全知识点抽象为可实践的挑战，参赛者通过理解与利用相关漏洞或机制完成解题。常见方向包括 Web、Pwn、Reverse、Crypto、Misc；线下决赛亦可能采用 AWD / AWDP、应急响应与综合渗透等形式。

技术方向

  • Web 网站漏洞挖掘与利用
  • Pwn 程序漏洞分析与利用
  • Reverse 软件逆向与算法还原
  • Crypto 密码分析与攻击
  • Misc 隐写、取证、流量与 OSINT
